# 坦克大战 (Battle City)

经典坦克大战游戏，使用 HTML5 Canvas 和 Web Audio API 制作。

## 功能特性

### 游戏玩法
- **3 个关卡**，难度递增
- **4 种敌人类型**：基础型、快速型、强力型、装甲型
- **6 种道具**：
  - ★ 星星 - 升级坦克（速度→射速→穿甲弹）
  - H 头盔 - 临时无敌
  - S 铲子 - 用钢墙强化基地
  - T 坦克 - 额外生命
  - ⏱ 计时器 - 冻结所有敌人
  - G 手榴弹 - 消灭屏幕上所有敌人

### 音频系统
- **背景音乐**：
  - 菜单音乐
  - 战斗音乐（循环播放）
  - 胜利音乐
  - 游戏结束音乐
  
- **音效**：
  - 射击音效
  - 爆炸音效（大/小）
  - 击中音效
  - 道具拾取音效
  - 敌人生成音效
  - 基地被毁音效

- **音频控制**：
  - **M 键** - 切换背景音乐开/关
  - **N 键** - 切换音效开/关
  - 侧边栏实时显示音频状态

## 操作说明

### 移动
- **方向键** 或 **WASD** - 控制坦克移动

### 射击
- **空格键** 或 **J 键** - 发射子弹

### 游戏控制
- **Enter** - 开始游戏 / 继续游戏
- **P** - 暂停 / 继续
- **M** - 音乐开关
- **N** - 音效开关

## 如何运行

1. 在浏览器中打开 `index.html`
2. 按 **Enter** 开始游戏
3. 消灭所有敌人，保护基地！

## 技术实现

- **HTML5 Canvas** - 游戏渲染
- **Web Audio API** - 8-bit 风格音乐和音效生成
- **纯 JavaScript** - 无外部依赖
- **像素风格** - 经典复古画面
